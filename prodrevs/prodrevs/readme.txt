== Description ==

Product Reviews creates custom post type (Products) and adds an option to display these in a widget, with filtering and categorization.
It relies on usage of ACF Lite.

Major features in Product Reviews include:

* Creates CPT Products (with predefined fields)
* Creates custom CPT taxonomy (Target Groups)
* Settings page allows for default target group to be selected for product display within widget
* Registers a widget

== Installation ==

Upload the Product Reviews plugin to your Wordpress website & Activate it.

== Frequently Asked Questions ==

= WHY I can't see products in the widget? =

Products need to be asigned to target groups first and default target group needs to be set.

= IS THE PLUGIN FREE? =

Yes.
